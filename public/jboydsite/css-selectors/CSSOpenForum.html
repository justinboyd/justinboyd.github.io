CSS Selectors

Show HTML


(V4): IE 8+
  • *
    Everything Selector

    This will target every single element.

    Example: *
  • #X
    ID Selector

    This will target element's by the id attribute.

    Example:
    #yeungling
  • .X
    Class Selector

    This will target element's by the class attribute.

    Example:
    .water
  • X
    Element Selector

    This will style a specific element type. Example: span, div, i, a

    Example:
    ul
  • X, Y
    Multiple Selector

    This will declare multiple selectors.

    Example:
    .beer, .wine
  • X Y
    Descendant Selector

    This will style all Y elements inside of an X element.

    Example:
    #liqour li
  • X > Y
    Direct Child Selector

    This will only select direct children.

    Example:
    #liqour > li
  • X + Y
    Adjacent Selector

    This will select the only element that is immediately preceded by the former element. Basically, it will get it's direct sibling.

    Example:
    .beer + li
  • X ~ Y
    Sibling Combinator Selector

    This is similar to the Adjacent Selector, but less strict. This will get any matching siblings.

    Example:
    .beer ~ li
  • X[title]
    Attributes Selector

    This will select any element with the attribute defined.

    Example:
    span[title]
  • X[href="foo"]
    More specific Attributes Selector

    This will select any element with the attribute value defined.

    Example:
    span[title="beer"]
  • X[href*="foo"]
    Regular Expression Attributes Selector

    This will select any element with the attribute value that contains "foo".

    Example:
    span[title*="liq"]
  • X:after, X:before
    Before and After Pseudo Elements

    These selectors use the "content" css property to add content to an element.

    Example:
    .jameson:before {
	content:"Oh ";
	background:green;
	color:#fff;
	border-radius:10px;
}
.jameson:after {
	content:", I heart you.";
	position:absolute;
	top:10px;
	background:green;
	color:#fff;
}
(V5) IE 9+
  • X:not(selector)
    Negation Pseudo Element

    This will select every element EXCEPT for the selector defined in the parenthesis.

    Example:
    .wine li:not(.savignon_blanc)
  • X:nth-child(n)
    nth-child Selector

    This will select the nth element in a selector. It takes numbers as values.

    Example:
    .wine li:nth-child(2)

    Or use the following to get every 2nd element:
    .wine li:nth-child(2n)
  • X:nth-last-child(n)
    nth-last-child Selector

    This will select the nth element from the end of a selector. It takes numbers as values.

    Example:
    .wine li:nth-last-child(2)
  • X:nth-of-type(n)
    nth-of-type Selector

    This will select the nth element of the selector type. It takes numbers as values.

    Example:
    .wine li:nth-of-type(3)
  • X:first-child
    first-child Selector

    This targets only the first child of the element's parent.

    Example:
    .wine li:first-child
  • X:last-child
    last-child Selector

    This targets only the last child of the element's parent.

    Example:
    .wine li:last-child
